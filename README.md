# Projeto 6 - Chatbot API Front-end

Projeto do programa desenvolve da Boticario, realizado por Valdomira Cardoso, Marcos Antônio Teixeira de Souza Junior, Fernando Matheus Mirinduba Dos Santos, Bárbara Machado de Araújo Gouveia e Thiago Silva Araujo. Esse repositório é referente a do front-end do projeto. Os responsaveis pala parte do front foram, Valdomira, Marcos Bárbara e Thiago, com Fernando sendo responsável pelo Back-end.


## Back-end
Construido Javascript e nodejs. 

Acesse o [back-end](https://github.com/Renchs/backend-p6-chatbot) do projeto.

### Atenção
Para utilziar do projeto e necessario ter uma key da api do Gemini, para ser direcionado para o site [clique aqui](https://console.cloud.google.com/). 
1. Faça o clone dos dois repositórios(back e front);
2. Coloque sua api key da gemini dentro do arquivo .env no back-end;
3. Rode o Back-end utilizando:

`npm run dev`

4. Em seguida abra o front-end com a extensão live-server ou similares, usando localhost.

## Front-end

O projeto foi pensado para simular um aplicativo de chat online(ex: Telegram, Whatsapp), aonde o usuário pode conversar com a AI do gemini de maneira mais similar a seu cotidiano.
O projeto também conta com outras funções como:
1. Salvar a conversa no local storage, com a possibilidade de deletar caso o usuário deseje;
2. Copiar mensagem do chatbot com um simples clique no ícone de copiar;
